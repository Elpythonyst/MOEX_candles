Данный репозиторий содержит в себе тестовое задание, заключавшееся в:
1. Обращении к API Московской Биржи с целью выгрузки дневных свечей по инструментам SBER, MOEX с начала 2013 года.
2. Расчете индикаторов SMA50, SMA200, RSI для данных из п.1
3. Визуализации индикаторов из п.2
4. Расчете коэффициента корреляции индикаторов из п.2, его визуализация и получение выводов.

Класс Stock с методами, выполняющими вышеперечисленные задачи, содержится в файле StockClass.py.
Все остальные файлы обращаются к нему и состоят буквально из двух строк.
Также оставляю для простых и быстрых экспериментов файл Ticker_info.ipynb.
Большого пространства для выводов я не вижу, т.к. к техническому анализу отношусь с некоторой долей скептицизма, но имеющиеся выводы находятся в файле conclusion.txt
